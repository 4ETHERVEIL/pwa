🫧 Bubblewrap (TWA) - Langkah Build

Cara build APK menggunakan Google Bubblewrap CLI

1
Install Node.js
2
Install Bubblewrap
3
Init Project
4
Build APK
Terminal Commands 
# 1. Install Node.js (https://nodejs.org)

# 2. Install Bubblewrap CLI globallynpm install -g @bubblewrap/cli

# 3. Inisialisasi project baru
bubblewrap init --manifest https://yourdomain.com/manifest.json

# 4. Build APK
bubblewrap build

# 5. Build App Bundle (untuk Play Store)
bubblewrap build --bundle

# Output: app-release-signed.apk 🎉
⚠️ Requirements
  • Java JDK 8+ terinstall
  • Android SDK (otomatis di-setup Bubblewrap)
  • Website harus HTTPS
  • manifest.json harus valid dan accessible
  • Service Worker harus aktif
🌐

🌐 PWABuilder.com - Langkah Build

Cara build APK menggunakan Microsoft PWABuilder

1
Deploy PWA
2
Visit PWABuilder
3
Download APK
Langkah-langkah 
1. Deploy PWA Anda ke hosting (Vercel, Netlify, GitHub Pages)
   Pastikan manifest.json & service-worker.js accessible

2. Buka https://pwabuilder.com

3. Masukkan URL website Anda → klik "Start"

4. PWABuilder akan analyze PWA Anda

5. Klik "Android" → pilih opsi packaging

6. Download APK/App Bundle yang sudah di-generate 🎉

⚡ Capacitor.js - Langkah Build

Cara build APK menggunakan Ionic Capacitor

1
Setup Project
2
Install Capacitor
3
Add Android
4
Build APK
Terminal Commands 
# 1. Buat project baru
mkdir my-pwa-app && cd my-pwa-app
npm init -y

# 2. Install Capacitor
npm install @capacitor/core @capacitor/cli

# 3. Inisialisasi Capacitor
npx cap init "My PWA App" com.mypwa.app

# 4. Buat folder www dengan file PWA Anda
mkdir www
# Copy index.html, manifest.json, sw.js ke folder www/

# 5. Tambahkan platform Android
npm install @capacitor/android
npx cap add android

# 6. Sync files ke Android
npx cap sync

# 7. Buka di Android Studio
npx cap open android

# 8. Build APK dari Android Studio:
# Build → Build Bundle(s) / APK(s) → Build APK 🎉
📱

📱 Apache Cordova - Langkah Build

Cara build APK menggunakan Apache Cordova

Terminal Commands 
# 1. Install Cordova CLI
npm install -g cordova

# 2. Buat project baru
cordova create mypwa com.mypwa.app "My PWA App"
cd mypwa

# 3. Tambahkan platform Android
cordova platform add android

# 4. Copy file PWA ke folder www/
# (ganti semua file di www/ dengan file PWA Anda)

# 5. Install plugin inappbrowser
cordova plugin add cordova-plugin-inappbrowser

# 6. Build APK
cordova build android

# 7. Build release APK
cordova build android --release

# Output: platforms/android/app/build/outputs/apk/release/app-release-unsigned.apk 🎉
